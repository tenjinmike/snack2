# 로또 당첨 번호 생성기 (Lotto Winner Generator) 프로젝트

## 1. 개요

이 프로젝트의 목표는 사용자에게 행운을 가져다줄 로또 번호를 생성하는 **로또 당첨 번호 생성기**를 만드는 것입니다. 고전적인 번호 추출 방식을 현대적이고 고급스러운 디자인으로 재현하며, 실제 로또 공이 굴러가는 듯한 애니메이션 효과를 제공합니다.

## 2. 기능

*   **번호 생성:** 1부터 45까지의 숫자 중 중복되지 않는 6개의 번호를 생성합니다.
*   **다중 행 생성:** 한 번에 5게임(5줄)의 번호를 생성하여 실제 로또 용지와 유사한 경험을 제공합니다.
*   **애니메이션 효과:** 번호가 하나씩 순차적으로 나타나는 부드러운 애니메이션을 적용합니다.
*   **번호별 색상:** 실제 로또 공의 색상 규칙(1-10: 노랑, 11-20: 파랑 등)을 적용하여 시각적 완성도를 높입니다.
*   **테마 지원:** 다크 모드와 라이트 모드를 지원하여 가독성을 높입니다.

## 3. 디자인 및 스타일

*   **메인 테마:** "골드 & 프리미엄" 스타일로 당첨의 기운을 전달합니다.
*   **시각 효과:** 
    *   번호 공(Ball)에 입체감 있는 그라데이션과 그림자를 적용합니다.
    *   생성 버튼에 "글로우" 효과와 클릭 애니메이션을 추가합니다.
*   **반응형:** 모바일에서도 실제 로또 용지를 보는 듯한 최적화된 레이아웃을 제공합니다.

## 4. 구현 단계

1.  **HTML (`index.html`):** 로또 공이 표시될 컨테이너와 생성 버튼 구조 설정.
2.  **CSS (`style.css`):** 로또 공 테두리, 색상, 애니메이션 및 테마 스타일링.
3.  **JavaScript (`main.js`):** 1-45 랜덤 번호 추출 로직 및 시각적 표시 기능 구현.
4.  **GitHub 업로드:** 수정된 코드를 `https://github.com/tenjinmike/snack2` 저장소에 커밋 및 푸시.
