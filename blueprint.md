# Snack 2 (현대적인 스네이크 게임) 프로젝트

## 1. 개요

이 프로젝트의 목표는 고전적인 '스네이크(Snake)' 게임을 현대적인 웹 기술로 재해석하여, 시각적으로 아름답고 재미있는 **Snack 2** 게임을 만드는 것입니다. 프레임워크 없이 HTML, CSS, JavaScript를 사용하여 구축하며, 부드러운 애니메이션과 반응형 디자인을 제공합니다.

## 2. 기능

*   **게임 플레이:** 방향키 또는 터치 스와이프로 스네이크를 조종하여 먹이(Snack)를 먹습니다.
*   **점수 시스템:** 먹이를 먹을 때마다 점수가 올라가고 스네이크의 길이가 길어집니다.
*   **난이도 조절:** 게임이 진행됨에 따라 속도가 점진적으로 증가합니다.
*   **최고 점수:** 로컬 스토리지를 사용하여 최고 점수를 저장하고 표시합니다.
*   **반응형 디자인:** 데스크톱과 모바일 기기 모두에서 최적화된 화면을 제공합니다.

## 3. 디자인 및 스타일

*   **색상 팔레트:** 생동감 있는 네온 스타일 또는 현대적인 파스텔톤을 사용하여 에너지를 전달합니다.
*   **애니메이션:** 스네이크의 이동과 먹이 획득 시 부드러운 트랜지션 및 효과를 적용합니다.
*   **레이아웃:** 중앙 집중형 게임 보드와 점수판을 배치합니다.
*   **효과:**
    *   글로우(Glow) 효과를 사용하여 게임 요소에 생동감을 더합니다.
    *   미묘한 배경 텍스처와 깊이감 있는 그림자를 적용합니다.

## 4. 구현 단계

1.  **HTML (`index.html`):** 게임 캔버스(Canvas)와 UI 요소(점수, 시작 버튼) 구조 설정.
2.  **CSS (`style.css`):** 게임 보드 스타일링 및 현대적인 UI 디자인 적용.
3.  **JavaScript (`main.js`):** 게임 루프, 스네이크 로직, 충돌 감지 및 점수 관리 구현.
4.  **GitHub 업로드:** 수정된 코드를 `https://github.com/tenjinmike/snack2` 저장소에 커밋 및 푸시.
