# 로또 번호 생성기 프로젝트

## 1. 개요

이 프로젝트의 목표는 사용자가 로또 번호를 생성하고 볼 수 있는 간단하고 매력적인 웹 애플리케이션을 만드는 것입니다. 현대적인 웹 기술(HTML, CSS, JavaScript)을 사용하여 프레임워크 없이 구축하며, 뛰어난 시각적 디자인과 사용자 경험에 중점을 둡니다.

## 2. 기능

*   **번호 생성:** 1부터 45까지의 숫자 중 6개의 고유한 랜덤 번호를 생성합니다.
*   **번호 표시:** 생성된 번호를 시각적으로 명확하고 아름답게 표시합니다.
*   **재생성:** 버튼 클릭 한 번으로 새로운 번호 세트를 생성할 수 있습니다.
*   **반응형 디자인:** 모든 화면 크기(데스크톱, 모바일)에서 완벽하게 작동합니다.

## 3. 디자인 및 스타일

*   **색상:** 활기차고 생동감 있는 색상 팔레트를 사용하여 에너지를 전달합니다.
*   **폰트:** 가독성 높고 표현력 있는 타이포그래피를 사용합니다.
*   **레이아웃:** 깨끗하고 균형 잡힌 레이아웃과 충분한 여백을 사용합니다.
*   **효과:**
    *   배경에 미묘한 노이즈 텍스처를 추가하여 고급스러운 느낌을 줍니다.
    *   번호 카드와 버튼에 부드러운 그림자(drop shadow)를 적용하여 입체감을 줍니다.
    *   인터랙티브 요소에는 "글로우" 효과를 추가하여 사용자의 상호작용을 유도합니다.

## 4. 현재 구현 계획

1.  **HTML (`index.html`):** 기본 구조 설정 (제목, 번호 컨테이너, 생성 버튼).
2.  **CSS (`style.css`):** 위에서 정의한 디자인 가이드라인에 따라 스타일링.
3.  **JavaScript (`main.js`):** 로또 번호 생성 로직 및 DOM 조작 기능 구현.
4.  **GitHub 업로드:** 현재 코드를 `https://github.com/tenjinmike/snack2` 저장소에 업로드.

## 5. 실행 단계

1.  `blueprint.md` 업데이트.
2.  수정된 모든 파일을 스테이징 (`git add .`).
3.  커밋 메시지와 함께 변경 사항 커밋.
4.  원격 저장소(`https://github.com/tenjinmike/snack2`)를 `origin`으로 추가.
5.  `main` 브랜치를 원격 저장소로 푸시.
